
 Logistic Regression Project



This project demonstrates a simple yet **unique** implementation of Logistic Regression using Python and Scikit-Learn.

It includes modular scripts for:
- Data Preparation
- Model Training
- Evaluation
- Visualization

#run this like
```bash
pip install -r requirements.txt
python src/prepare_data.py
python src/train_model.py
python src/evaluate_model.py
python src/visualize_results.py
